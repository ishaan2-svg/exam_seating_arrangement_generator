Two-Factor Authentication Required

Important: Teachers must set up 2FA for security. After registration, you'll receive a QR code to scan with Google Authenticator or similar app.

Setup Instructions:

  1. Install Google Authenticator or Authy on your phone
  2. After registration, scan the QR code provided
  3. Use the generated codes for admin panel access only
  4. Regular login won't require 2FA codes

Student Registration Process

How it works:

  1. Enter your Student ID exactly as it appears in school records
  2. System will verify your ID against student database
  3. Your Student ID will be used as your username (URL-safe)
  4. If not found, contact admin to add your information first
  5. No 2FA setup required for students
