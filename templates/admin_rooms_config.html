Room Configuration Guidelines

  • Capacity: Should match the actual physical seats available in the room
  • Layout: Columns × Rows should be ≥ Capacity for proper visualization
  • Constraints: Lower max subjects/branches values help reduce exam conflicts
  • Year Restrictions: Help organize exams by academic level and reduce conflicts
  • Branch Restrictions: Can be used to separate different departments if needed
